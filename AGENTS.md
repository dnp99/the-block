# AGENTS.md

This project's agent guidance is maintained in **[CLAUDE.md](CLAUDE.md)** — the single
canonical set of instructions for any AI coding agent working in this repo.

See `CLAUDE.md` for:

- Stack and project structure
- Design-system rules (and the canonical spec in `docs/DESIGN-SYSTEM.md`)
- AI integration conventions (server-only key, prompt location, structured-output validation, fallbacks)
- Error-handling tiers and bid rules
- The mandatory pre-commit checklist

There is intentionally no duplicate rule set here — keep all agent guidance in `CLAUDE.md`
so the two never drift.
